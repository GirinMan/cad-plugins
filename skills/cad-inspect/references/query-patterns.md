# CAD Inspect Query Patterns

Useful targeted inspection prompts:

- "Find labels containing pump / PUMP / 펌프"
- "Show text on layer MAP-HOUSE"
- "Summarize block names and layers"
- "Extract elevation labels like EL=..."
- "Find pipe diameter labels like Ø 300"
- "List layers with the most text"

Prefer filtered output over full-file dumps.
